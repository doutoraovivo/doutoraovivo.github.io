Login Programático

Para aproveitar ao máximo da sua integração whitelabel, você pode integrar nosso Consultorio SaaS dentro do seu sistema, sem precisar que seu cliente acesse dois sistemas direfentes, e com isso, ter dois logins.

Com o Login programático você faz uma chamada de API para criar um 'passe'. Com esse 'passe', você aciona a url do consultorio com uma rota especial, a rota de PSO.

Confira os passos:

1. Criação do Passe

Para a criação do passe é necessário ter previamente ter incluído seu cliente na DaV. A chamada é feita na API de Credential passando o person ID. O passe funciona por 6 horas.

POST /credential/pso/person/<person_id>

Sugestão: Para uma melhor performance pense em chamar de forma assíncrona, a execução desse serviço em momento de login do seu cliente em seu sistema.

2. Abertura do Front

Para a execução do front dentro do seu sistema usando o passe de cliente, incorpore na sua página um iframe sem bordas com a URL abaixo.

<iframe src="https://<seu dominio dav>.dav.med.br/pso/<passe id>"></iframe>

Sugestão: Adeque as cores do consultório para se adequar às cores do seu sistema.
